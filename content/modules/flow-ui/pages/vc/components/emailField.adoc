== Основы

Компонент `emailField` - это специализированное поле ввода, разработанное для сбора и проверки адресов электронной почты. Оно расширяет базовый компонент `textField`, добавляя специальные функции и проверки, адаптированные для адресов электронной почты.

== Привязка данных

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.ru/ui-samples/sample/email-field-dataaware" class="live-demo-btn" target="_blank">ДЕМОНСТРАЦИЯ</a>
</div>
++++

Привязка данных означает связывание визуального компонента с xref:data/data-containers.adoc[контейнером данных]. Изменения в визуальном компоненте или соответствующем контейнере данных могут вызывать обновления друг друга. Смотрите xref:flow-ui:data/data-examples.adoc[Использование компонентов данных] для получения дополнительной информации.

Следующий пример создает `emailField`, связанный с данными.

[source,xml,indent=0]
----
include::example$/onboarding/src/main/resources/com/company/onboarding/view/component/emailfield/emailField-view.xml[tags=data;layout;data-binding]
----
<1> `InstanceContainer` для сущности `Customer`.
<2> Встроенный план выборки экземпляра сущности, находящегося в контейнере.
<3> Привязка компонента к контейнеру данных и свойству. Атрибут `dataContainer` содержит ссылку на контейнер данных `customerDc`, а атрибут `property` ссылается на атрибут сущности `email`.

== Варианты Оформления

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://demo.jmix.ru/ui-samples/sample/email-field-theme-variant" class="live-demo-btn" target="_blank">ДЕМОНСТРАЦИЯ</a>

Атрибут xref:vc/common-attributes.adoc#themeNames[themeNames] позволяет вам назначить конкретный стиль `emailField` из набора предопределенных вариантов.

`emailField` имеет несколько вариантов оформления:

* `small`
* `align-left`
* `align-center`
* `align-right`
* `helper-above-field`

Более подробное описание с примерами вы найдете в xref:vc/components/textField.adoc#themeVariants[соответствующем разделе] для компонента `textField`.
