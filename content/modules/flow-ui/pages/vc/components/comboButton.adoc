`comboButton` 是带有下拉列表的按钮，可以提供一组不同的 xref:actions/declarative-actions.adoc[操作]。

该组件与 xref:vc/components/dropdownButton.adoc[dropdownButton] 很类似。

主要区别是该组件按钮本身也可以添加操作。

== XML 属性

在 Jmix 中，所有组件都有一些作用一致的 xref:vc/common-attributes.adoc[通用属性]。
下面是 `comboButton` 的特殊属性：

[cols="1h,5,1", grid=rows, frame=none]
|===
|*名称*
|*描述*
|*默认值*
|[[dropdownIcon]]dropdownIcon
|设置 `comboButton` 的下拉图标。
|`VaadinIcon.CHEVRON_DOWN`
|===
