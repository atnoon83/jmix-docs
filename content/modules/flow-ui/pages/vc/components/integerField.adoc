== 步长

++++
<div class="jmix-ui-live-demo-container">
    <a href="https://vaadin.com/docs/latest/components/number-field#step" class="vaadin-docs-btn" target="_blank">Vaadin</a>
</div>
++++

`step`（步长）属性用于控制用户使用向上/向下箭头（也称为步长按钮）交互时的增减值。步长决定每次交互时值的变化量。可以用来定义 `integerField` 的精度级别。

NOTE: `step` 属性通常与 <<min,min>> 和 <<max,max>> 属性结合使用，以定义数字的有效范围。

`stepButtonsVisible` 属性控制是否显示步长按钮。

== XML 属性

在 Jmix 中，所有组件都有一些作用一致的 xref:vc/common-attributes.adoc[通用属性]。
下面是 `integerField` 的特殊属性：

[cols="1h,5,1", grid=rows, frame=none]
|===
|*名称*
|*描述*
|*默认值*
|[[max]]max
|指定允许输入的最大值。大于 `max` 的值会导致验证失败。
|
|[[min]]min
|指定允许输入的最小值。小于 `min` 的值会导致验证失败。
|
|[[step]]step
|设置控件的值变化量。设置当点击增加/减少按钮时，数字变化的步长。参阅 <<about-step, 步长>>。
|`1`
|[[stepButtonsVisible]]stepButtonsVisible
|设置步长按钮的可见性。参阅 <<about-step, 步长>>。
|`false`
|===
