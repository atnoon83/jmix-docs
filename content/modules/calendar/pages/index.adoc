= Calendar

The Calendar add-on provides integration with https://fullcalendar.io/[FullCalendar] JavaScript component. The add-on contains `FullCalendar` component that allows you to show events in a calendar with various views (day, week, month, etc.) and edit them by dragging and resizing.
